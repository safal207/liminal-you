# Liminal-You

> Социальный слой LIMINAL OS — “Ты”.  
> Пространство, где внутренние миры встречаются через резонанс.

### Stack
- FastAPI
- React + TypeScript
- PostgreSQL
- Docker-compose

### Run

```bash
docker-compose up --build
```

Frontend → http://localhost:5173  
Backend → http://localhost:8000/docs
